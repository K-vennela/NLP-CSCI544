John,

in order to upgrade to the latest version, you will need to download from the Support Portal and install on your servers, IN THIS ORDER:

1) rcs-setup-9.0.0.exe

2) rcs-setup-9.1.0.exe

3) rcs-setup-9.1.1.exe

Each file will have to be run ON THE BACKEND FIRST, AND ON THE COLLECTOR AFTERWARDS.

You can use the attached license file.

After installing successfully rcs-setup-9.1.1.exe, you can download and install rcs-ocr-9.1.1 on the backend only.

After downloading each file, please make sure that it is not corrupted by checking the md5 checksum:

rcs-ocr-9.0.0.exe => 52c623779e8a5c1036d0b67872a3da6b

rcs-setup-9.1.0.exe => 941e7a5ae6b7343523a5afb7197b413b

rcs-setup-9.1.1.exe => 6e2d403e6bce68a889695429c1da61ea
